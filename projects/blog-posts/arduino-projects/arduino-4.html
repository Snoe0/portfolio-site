Arduino HW 4 - Color Mixing Lamp

IGME 470

This project was our first time using RGB LEDs and phototransistors to read light values. As my fourth homework assignment, I am still getting used to reading the resistor values and learning how to troubleshoot, but it worked out.

I often do work under a lamp, so initially my LED wasn't lighting up, however I eventually discovered that the light in my room was simply too dim. By using my phone's flashlight, I was able to get the LED to light up.

Overall, this project was a fun way to change an LED using varying levels of Red Green and Blue.
